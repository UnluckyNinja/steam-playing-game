# Steam Playing Game

Automatically generates status message based on your steam status. 

全自动生成Steam游戏状态信息  
在OBS添加网页来源即可 （[http://steam-playing-game.vercel.app/]）  
页面内鼠标悬浮即可查看说明  

See details in [http://steam-playing-game.vercel.app/]
